# CRUD Application using ADO.NET

This is an ASP.NET Core MVC Web Application performing CRUD (Create, Read, Update, Delete) operations using ADO.NET and SQL Server.

## Features
- Create new records
- Read/View list of records
- Edit/Update existing records
- Delete records

## Tech Stack
- C# / ASP.NET Core MVC
- ADO.NET
- SQL Server
- HTML / CSS / Bootstrap
